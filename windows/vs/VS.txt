vs快捷键支持自定义（tools keyboard 里面搜索找到相应的指令进行设置快捷键）
光标所在的行，用ctrl + c就能复制整行了

CTRL + M, CTRL + O折叠代码定义

CTRL + M, CTRL + L展开代码定义

ALT  + M,  打开函数列表

SHIFT + ALT +　Ｏ　　打开文件

ALT + O  打开对应cpp文件或者.h文件

ALT　＋　Ｇ　　查看调用

一般需要设置的还有：查看引用，向前，向后

在工程属性中可以设置很多属性，比如全局宏，连接设置，服务器，端口，等等


vs 配置有两套，工程中一套，配置文件中一套，他们会在一定的时机进行同步
配置文件修改，vs会马上会进行同步
vs工程中的配置修改，只有运行时才会真正保存到配置文件中，进行同步


.sln 是负责整个解决方案的，包含多个.vcxproj
.vcxproj是负责相应工程的

各个功能窗口在view中设置隐藏或出现
